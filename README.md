O imtuito é criar uma agente de IA que fará uma entrevista com a descrição de uma vaga de emprego com o cargo, nível, tecnológicas, soft skills
e posteriormente ela fará uma entrevista com base nesses dados fornecidos pelo usuário.

a Inteligência artifical utilizada foi o copilot IA conforme a instrução do mentor henrique silva da digital inovation one, sendo um projeto 
dentro do bootcamp da caixa.

English version

The immediate task is to create an AI agent that conducts an interview based on a job description, including position, level, technology, and interpersonal skills.

Then, the agent will conduct an interview based on the data provided to the user.

The artificial intelligence used was Copilot AI, following instructions from mentor Henrique Silva of Digital Innovation One, and is a project from Caixa's bootcamp.
